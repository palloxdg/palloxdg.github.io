Quick Start

Up and running
in 5 minutes

Everything you need to get Fluxee posting your tip menu and thanking your fans — from zero to live.

1
🧩 Install Fluxee from the Chrome Web Store

Fluxee is a Chrome extension — no separate app to install, no account to create. Click the button below, then click Add to Chrome.

Install Fluxee — Free
Suite tier only: also download and run FluxeeBridge — the local relay app that powers OBS overlays. Extract the ZIP and run FluxeeBridge.exe once. If Windows shows a security warning click More info → Run anyway.
2
📌 Pin and open Fluxee

Click the puzzle icon 🧩 in the Chrome toolbar, find Fluxee, and click the pin icon to keep it visible. Then click the Fluxee icon to open the sidepanel.

3
🔑 Activate your license Pro & Suite only

Lite is free — skip this step. If you purchased Pro or Suite, click the License indicator in the status bar at the top of the sidepanel, enter the license key from your confirmation email, and click Activate. The indicator turns green when active.

4
🟢 Check the API indicator

Open your Fansly stream page in Chrome, then open the Fluxee sidepanel. The API indicator should turn green — this confirms Fluxee can see your stream and is ready to work.

If API stays grey, make sure you are on your Fansly stream page — not the dashboard or another tab.

5
📋 Create your first tip menu

In the sidepanel, type or paste your tip menu into the Tip Menu Content field. One item per line, for example:

Dance — $10
Sing a Song — $20
Full Show — $50

Then click Save Menu. Give it a name when prompted. You can create multiple menus and switch between them at any time.

6
🚀 Start the bot

Click Start Bot. Fluxee will post your active menu to chat on the interval you set, and automatically thank every tipper by name and item. The button turns red — that means it's live.

That's it. Your stream is now running on autopilot. 🎉

✅

You're all set

Fluxee is live. Explore the full guide whenever you're ready to unlock everything else.
